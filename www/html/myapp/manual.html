Návod na použitie

V tejto aplikácii môžete upravovať PDF dokumenty rôznymi spôsobmi:

  • Spájanie PDF
  • Odstraňovanie strán
  • Pridávanie vodoznakov
  • Šifrovanie a dešifrovanie

Použitie cez frontend

Každá funkcia má vlastné rozhranie, kde si môžete vybrať PDF a nastaviť možnosti.

Použitie cez API

Ak ste prihlásený a máte API kľúč, môžete volat funkcie cez HTTP požiadavky.
